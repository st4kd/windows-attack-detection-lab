# Windows Attack Detection Lab

## Overview

This project demonstrates the creation and investigation of simulated Windows attack activity using Sysmon and Splunk.

The goal is to generate realistic security telemetry, develop detections, investigate suspicious activity, and document the findings using a structured incident-response approach.

## Objectives

- Build a Windows security monitoring lab
- Configure Sysmon logging
- Ingest Windows telemetry into Splunk
- Generate safe simulated attack activity
- Write detection queries
- Investigate suspicious process and command activity
- Map detections to MITRE ATT&CK

## Planned Detection Scenarios

- Suspicious PowerShell execution
- Encoded PowerShell commands
- Suspicious process relationships
- Credential access indicators
- Persistence activity
